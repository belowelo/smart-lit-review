# smart-lit-review

a batch tool to automate your lit review from title + abstract + keywords! 
this tool uses the semantic scholar API to fetch titles, abstracts, and semantic scholar pages of publications relevant to your paper.
i wouldn't use this as your primary lit review tool, but this works great as a secondary layer to catch things that you might have missed! simply download both files, place them in a clean folder, then run 'start here!.bat'!
